# SORKIN-2 exploratory workspace

`explore/` is a sandbox for speculative SORKIN-2 ideas. It is for hypothesis
generation, not confirmed claims.

Speculative ideas are welcome here, including:

- numerical resonances
- algorithmic analogies
- topology heuristics
- unconventional causal-set intuitions
- gamma/T0 schedule hypotheses
- landscape-barrier hypotheses

Repository responsibilities are separated as follows:

- `docs/` contains conservative interpretation and stable documentation.
- `benchmarks/` contains reproducible outputs and benchmark panels.
- `explore/` contains speculative notes and hypothesis generators.

Working cycle:

wild idea -> explore note -> minimal test -> conservative interpretation if it survives

## Current exploratory directions

- gamma resonance and thermal half-life hypotheses
- T0/gamma schedule accessibility
- sparse, hub, layered, and sprinkling N=12 topologies
- combinatorial rigidity of causal orders
- algorithmic recoverability boundaries
- possible landscape barriers in the Bombelli annealer

## Guardrails

- `explore/` is not a proof space.
- `explore/` is not publication-ready documentation.
- `explore/` must not contaminate benchmark claims.
- Nothing in `explore/` should be cited as evidence unless later tested.
- Every promoted claim must be backed by a named run, file, or reproducible diagnostic.
- A hypothesis may move from `explore/` to `docs/` only after a minimal targeted test survives.
