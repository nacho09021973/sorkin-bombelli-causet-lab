# N=20 Known-truth Validation

This probe validates the unchanged historical Bombelli annealer on one
manufactured known-truth Minkowski sprinkling case:

- `N = 20`
- `family = minkowski`
- `d_spacetime = 2`
- `case_seed = 1959`
- `schedule_label = gamma_0p5`
- `cooling_factor = 0.5`
- `budget_label = medium_25_25_8`
- `optimizer_seeds = 1959, 1962, 1987, 2001`

The known-truth case is generated by:

```bash
python3 explore/known_truth_n20/generate_known_truth_n20.py
```

The validation probe is run with:

```bash
python3 explore/n20_known_truth_validation/run_n20_known_truth_validation.py
```

It records one row per annealing block using read-only `block_callback` access
to `sim.rold`/`sim.xold`, then compares the endpoint with the best checkpoint
seen by oracle causal F1.

This is a SORKIN-2 diagnostic. It does not change `cones.py`,
`validation_suite.py`, the historical objective, the move set, or the
annealer dynamics. It is not a community golden table and should not be read
as an embeddability, physical-gamma, or N-transition claim.
