# Exploratory Phase Archive

This directory preserves exploratory phase material that is outside the active
SORKIN-2 scope.

Archived here:

- `benchmarks/foundation/phase1e_*`
- `benchmarks/foundation/phase2g_*`
- `benchmarks/foundation/phase3*`
- `benchmarks/foundation/phase4a_*`
- `benchmarks/foundation/phase4b_*`
- `benchmarks/foundation/phase5_*`
- corresponding `tools/build_phase1e_*`, `tools/build_phase2g_*`,
  `tools/build_phase3*`, `tools/build_phase4a_*`, `tools/build_phase4b_*`,
  and `tools/build_phase5_*` scripts
- corresponding Phase4B and Phase5 tests

These files are retained for provenance. They should not be used as active
claims for SORKIN-2 without an explicit decision and a new audit.

SORKIN-2 does not currently pursue Phase3 symbolic regression, Phase4A/4B
survival or visual sweeps, or Phase5 morphology. The active question is
algorithmic recoverability of known-truth causal realizations by the historical
Bombelli annealer.
