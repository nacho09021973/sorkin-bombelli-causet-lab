# SORKIN-2 Known-Truth Inputs

This directory contains permanent input files for minimal SORKIN-2
known-truth sanity checks.

`chain_4_d2.in` is a four-element total chain. `antichain_4_d2.in` is a
four-element antichain with no causal relations. Both are guaranteed
embeddable in 1+1 Minkowski spacetime by elementary construction.

These inputs do not prove manifoldlikeness, do not test general
embeddability, and do not prove completeness of the Bombelli annealer. They
are only trivial known-truth sanity checks for later recoverability runs.

These cases have not yet been executed as SORKIN-2 runs. No result CSV, JSON,
text output, or graphical output is implied by the presence of these input
files.
